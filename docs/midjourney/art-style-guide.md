# Zombie Game Art Style Guide

## Chosen Style
**Cartoon / Plants vs Zombies aesthetic**

## Base Style Prompt (append to all prompts)
```
, cartoon style, Plants vs Zombies aesthetic, bright saturated colors, friendly exaggerated features, soft rounded shapes, clean bold outlines, no blood or gore, suitable for children, white background --ar 1:1 --style raw
```

## Character Template
```
A cute cartoon [CHARACTER], [SPECIFIC FEATURES], big round expressive eyes, goofy friendly expression, pastel [COLOR] skin/body, wearing [OUTFIT], cartoon style, Plants vs Zombies aesthetic, bright saturated colors, friendly exaggerated features, soft rounded shapes, clean bold outlines, no blood or gore, suitable for children, white background --ar 1:1 --style raw
```

### Cartoon Style Example
```
A cute cartoon zombie character for a children's game, bright saturated colors, friendly exaggerated features, big round eyes with goofy expression, pastel green skin, soft rounded shapes, no blood or gore, silly smile showing two square teeth, wearing torn but colorful clothes, white background, clean lines, similar to Plants vs Zombies aesthetic --ar 1:1 --style raw
```


## Asset Categories

### Zombies
- Basic zombie
- Fast zombie
- Tank zombie
- Special zombies (themed variations)

### Player Character
- Kid hero (boy/girl variants)
- Different outfits/costumes

### Items & Power-ups
- Weapons (water guns, foam launchers, etc.)
- Power-ups (speed boost, shield, etc.)
- Collectibles

### UI Elements
- Health icons
- Score elements
- Buttons

### Backgrounds
- Use `--ar 16:9` for backgrounds instead of 1:1

## Color Palette
| Element | Colors |
|---------|--------|
| Zombies | Pastel green, purple accents |
| Heroes | Bright, warm colors |
| Power-ups | Yellow, gold, glowing |
| Backgrounds | Soft pastels, not distracting |
