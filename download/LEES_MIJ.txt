========================================
    ZOMBIE APOCALYPSE - Python Spel
========================================

Welkom bij het Zombie Apocalypse spel!
Hier leer je Python programmeren door een spannend zombie spel te maken.


HOE TE BEGINNEN
---------------

1. Open een terminal (Command Prompt of PowerShell)

2. Installeer wat je nodig hebt:

   pip install pgzero

3. Ga naar een level map, bijvoorbeeld:

   cd levels/level-1

4. Start het spel:

   python zombie.py

   (Voor level 4.5 en 5 gebruik je: pgzrun zombie.py)


DE LEVELS
---------

level-1/  ->  Je eerste keuzes maken
level-2/  ->  Loops: het spel blijft doorgaan
level-3/  ->  Lijsten: items en inventory
level-4/  ->  Functies: code opdelen
level-4.5/ -> Pygame Zero: graphics!
level-5/  ->  Het complete zombie spel
level-6/  ->  Bouw je eigen spel! (starter kits)
